说到数据描述，我们最先想到的应该就是pandas.describe函数，它可以给出常规的数据统计，但其实并不详细。


今天我们要说的是“一键数据描述”：pandas-profiling。它的用法简单到让你不可思议：

df.profile_report(style={'full_width':True})

然后展现的结果就如下图所示


还可以一键生成Html文件保存，随时随地都可以打开浏览

profile = data.profile_report(title='soton Dataset')
profile.to_file(output_file='result/soton_report.html')


当然，如果你习惯了pycharm，这个还提供了对应的插件，简直强大到没朋友！
